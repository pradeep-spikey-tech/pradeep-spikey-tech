Background & Journey

My path into software development began through self-directed learning in 2013. Initially attending University at Buffalo (SUNY) for Mechanical Engineering, I discovered my passion for programming and chose to pursue it independently. I immersed myself in iOS development, learning Swift, programming principles, Core Data, and cloud synchronization technologies.

In 2014, I built and released my first application, Everything Journal, to the App Store. Originally created as a personal productivity tool using iOS Core Data and iCloud sync, this project became the foundation of my professional career. It opened doors to freelance opportunities, contract work with development agencies, and eventually founding Spikey Tech Inc. Over a decade later, I'm rebuilding Everything Journal with modern technologies as a flagship product for 2025.
